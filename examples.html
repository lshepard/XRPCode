XRP Code Examples

Copy these examples into the Code Editor to get started with your XRP robot.

Joystick Control

Control your XRP with a gamepad using arcade-style driving. Connect via Bluetooth, then use the left joystick to drive forward/backward and turn.

joystick_control.py 
from XRPLib.defaults import *
from XRPLib.gamepad import *
import time

# Get the default gamepad instance
gp = Gamepad.get_default_gamepad()
print("Starting!")

while True:
    # Arcade drive: one joystick controls everything
    throttle = gp.get_value(gp.Y1)  # Forward/backward on left stick
    rotation = gp.get_value(gp.X1)  # Turn left/right on left stick

    # Use arcade method for intuitive control
    drivetrain.arcade(throttle, rotation)
    time.sleep(0.2)

Servo Control

Control a servo motor connected to your XRP. Make sure you have a servo connected to the Servo 1 port as shown in the building instructions. Servos are useful for building arms, grippers, and other mechanisms. Learn more about building an arm in the XRP User's Guide.

servo_control.py 
from XRPLib.defaults import *
import time

# servo_one is available from defaults
# You can also use servo_two, servo_three, servo_four

# Move servo to different positions
servo_one.set_angle(0)    # Move to 0 degrees
time.sleep(1)

servo_one.set_angle(90)   # Move to 90 degrees
time.sleep(1)

servo_one.set_angle(180)  # Move to 180 degrees
time.sleep(1)

# Return to starting position
servo_one.set_angle(0)

# Use servo_one.free() to let the servo spin freely

Servo with Gamepad

Control a servo using gamepad buttons while driving. Press A to move one direction, B for the other, or release both for neutral.

servo_gamepad.py 
from XRPLib.defaults import *
from XRPLib.gamepad import *
import time

gp = Gamepad.get_default_gamepad()
print("Starting!")

while True:
    # Arcade drive with left stick
    throttle = gp.get_value(gp.Y1)
    rotation = gp.get_value(gp.X1)
    drivetrain.arcade(throttle, rotation)

    # Control servo with buttons
    if gp.is_button_pressed(gp.BUTTON_A):
        servo_one.set_angle(180)
    elif gp.is_button_pressed(gp.BUTTON_B):
        servo_one.set_angle(0)
    else:
        servo_one.set_angle(100)  # Neutral position

    time.sleep(0.2)
