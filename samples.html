üöó Sample 1: Drive Straight and Back

drive_straight_back.py

This program makes the robot drive forward 20 cm, pause, then drive backward 20 cm to return to the starting position.

üí° Tip: Change the distance value (20) to make the robot drive further or shorter! 
# Drive Straight and Back
# Robot drives forward, pauses, then returns

from XRPLib.defaults import *
import time

# Distance to travel (in centimeters)
distance = 20

# Speed (0.0 to 1.0, where 1.0 is full speed)
speed = 0.5

# Drive forward
print("Driving forward...")
drivetrain.straight(distance, max_effort=speed)

# Pause for 1 second
print("Pausing...")
time.sleep(1)

# Drive backward (negative distance)
print("Driving backward...")
drivetrain.straight(-distance, max_effort=speed)

# Done!
print("Back to start!")

üéÆ Sample 2: Arcade Joystick Control

arcade_gamepad.py

Control your robot using a Bluetooth gamepad! The left joystick controls forward/backward movement, and left/right turning. Press the B button to stop the program.

üí° Tip: Make sure to pair your Bluetooth controller with the XRP before running. The left stick Y-axis controls speed, and X-axis controls turning. 
# Arcade Joystick Gamepad Control
# Control your robot with a Bluetooth gamepad

from XRPLib.defaults import *
from XRPLib.gamepad import Gamepad
import time

# Get the gamepad
gamepad = Gamepad.get_default_gamepad()

print("Gamepad connected!")
print("Use left joystick to drive")
print("Press B button to stop")

# Main control loop
while True:
    
    # Check if B button is pressed to exit
    if gamepad.is_button_pressed(Gamepad.BUTTON_B):
        print("Stopping...")
        break
    
    # Get joystick values
    # Y1 = forward/backward (-1 to 1)
    # X1 = left/right turn (-1 to 1)
    forward = gamepad.get_value(Gamepad.Y1)
    turn = gamepad.get_value(Gamepad.X1)
    
    # Apply arcade drive
    # forward: positive = forward, negative = backward
    # turn: positive = turn left, negative = turn right
    drivetrain.arcade(forward, turn)
    
    # Small delay to prevent overwhelming the robot
    time.sleep(0.05)

# Stop the robot and gamepad when done
drivetrain.stop()
gamepad.stop()
print("Done!")
